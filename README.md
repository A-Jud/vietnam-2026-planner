🇻🇳 וייטנאם 2026 - AI Travel Planner

ברוכים הבאים לאפליקציית "וייטנאם 2026 - Planner V2"!
זהו כלי חכם ומתקדם לתכנון טיולים, המשלב ניהול מסלול דינמי עם יכולות בינה מלאכותית (AI) מתקדמות של Google Gemini ו-OpenAI. האפליקציה מאפשרת לתכנן, לערוך ולשתף מסלולי טיול בקלות, תוך סנכרון מאובטח ל-Google Drive.

✨ פיצ'רים מרכזיים

🗺️ מפה אינטראקטיבית ודינמית: מפת Leaflet המציגה את המסלול שלך בזמן אמת. תמיכה ב-Geocoding חי הממקם ערים חדשות אוטומטית על המפה, וקיבוץ חכם (Clustering) של ימים מרובים באותו יעד.

📅 ניהול מסלול V2: ארכיטקטורה חכמה מבוססת מזהים ייחודיים (IDs). אפשרות קלה להוספה, עריכה, מחיקה והזזה של ימים במסלול. התאריכים מחושבים אוטומטית לפי תאריך ההתחלה!

🔄 גרירה והעברה (Relocation): העבר בקלות פעילויות, מיקומים וטיפים בין ימים שונים בלחיצת כפתור.

🤖 המלצות AI מתקדמות:

בניית מסלול שלם: תן ל-AI לבנות עבורך מסלול מרובה ימים מאפס, מותאם אישית לפי המוצא והסגנון שלך.

תכנון יום מחדש (Replan): רוצה לשנות תוכניות? ה-AI יתכנן מחדש יום ספציפי בהתאם לבקשותיך, תוך התחשבות ביעדים שלפני ואחרי.

מומחה מקומי: קבל טיפים סודיים, פנינות חבויות והמלצות קולינריות אותנטיות (🍜) לכל יעד.

🔗 קישורי מפות חכמים: המערכת מייצרת אוטומטית קישורי Google Maps רובאסטיים ומדויקים (לא עוד קישורים שבורים ב-iFrame!), כולל מנוע חיפוש פנימי למציאת מיקומים.

☁️ סנכרון גיבוי: גיבוי אוטומטי ל-Google Drive של המשתמש (באמצעות Firebase/Google Auth), בנוסף לאפשרויות ייבוא וייצוא מקומיות (JSON).

🔒 אבטחה: מפתחות ה-API שלך ל-Gemini/OpenAI מוצפנים בצד הלקוח באמצעות Web Crypto API (AES-GCM 256-bit).

📱 רספונסיביות מלאה: ממשק נוח המותאם לשימוש גם במחשב וגם במובייל, כולל אפשרות לשינוי גודל פאנל הצד (Resizer).

🚀 איך מתחילים?

אתחול: פתח את קובץ ה-HTML בדפדפן. האפליקציה תטען נתוני דמו התחלתיים של טיול בוייטנאם.

התחברות לענן (אופציונלי): התחבר עם חשבון ה-Google שלך כדי לאפשר שמירה אוטומטית וסנכרון בין מכשירים.

הגדרת AI: לחץ על סמל ההגדרות (⚙️), בחר את ספק ה-AI המועדף עליך (Gemini מומלץ) והכנס מפתח API. המפתח יוצפן ויישמר מקומית בדפדפן שלך בלבד.

התחל לתכנן! * ערוך ימים קיימים, הוסף חדשים, או בקש מה-AI לבנות עבורך מסלול.

גלה טיפים ואוכל מקומי, ושמור את המועדפים עליך לתוך המסלול.

לחץ על "נווט" כדי לקבל מסלול נסיעה יומי מחושב ב-Google Maps.

🛠️ טכנולוגיות

Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6 Modules)

Map: Leaflet.js, OpenStreetMap Nominatim (Geocoding)

Backend/Auth: Firebase (Auth, Firestore), Google Identity Services

AI Integrations: Google Gemini API, OpenAI API

Security: Web Crypto API (AES-GCM)
