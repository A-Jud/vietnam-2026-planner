## מתכנן הטיולים החכם (Smart Trip Planner)

מתכנן טיולים גנרי, חכם ומבוסס AI המאפשר למשתמשים לבנות, לערוך ולנהל מסלולי טיול מורכבים לכל יעד בעולם. המערכת משלבת ממשק משתמש אינטואיטיבי, מפה אינטראקטיבית (Leaflet), חיבור מובנה למפות גוגל (Google Maps), ומנועי בינה מלאכותית (OpenAI או Gemini) ליצירת המלצות מותאמות אישית.

## תכונות מרכזיות

- **בניית מסלול אוטומטית מבוססת AI:** יצירת מסלול שלם ממספר ימים מוגדר, כולל חלוקה לוגית של יעדים, אטרקציות ואפשרויות לינה.
- **עריכה ידנית וידידותית:** הוספה, עריכה ומחיקה של ימים, כולל ניהול יעדים מרכזיים, תיאור יומי, ופרטי לינה.
- **שדרוג יומי חכם (AI Magic):**
  - ✨ טיפים: המלצות למקומות סודיים, פנינים נסתרות ונקודות תצפית.
  - 🍜 אוכל מקומי: המלצות למנות מקומיות ומסעדות ספציפיות פעילות.
  - 🏨 מלונות: אפשרויות לינה מגוונות (תקציב, יוקרה, משפחות).
  - 🎯 אופציות למסלול: אפשרויות פעילות נוספות לבחירה.
  - 👷 נותני שירותים: מדריכים, נהגים וספקים מקומיים.
  - 🤖 תכנון מחדש: ג'ינרוט מסלול חדש לאותו היום על בסיס בקשה ספציפית.
- **ניווט חכם (Smart Routing):** יצירת מסלול Google Maps מרובה נקודות מתוך יעדים, אפשרויות נבחרות וטיפים — מוצג בתוך האפליקציה או בדפדפן חיצוני.
- **השוואת גרסאות במפה:** השוואת המסלול הנוכחי לגרסאות עבר על גבי מפת Leaflet (נוכחי כחול / עבר ירוק), וכן מעבר בין גרסאות ב-Google Maps.
- **מפה אינטראקטיבית מובנית:** תצוגה גיאוגרפית של מסלול הטיול עם פוליגונים וסמנים לכל נקודות העניין.
- **היסטוריית גרסאות:** כל שינוי (הוספת טיפ, אפשרות, מלון, ספק, תכנון מחדש) יוצר גרסה חדשה בהיסטוריה עם אפשרות שחזור.
- **גיבוי ענן (Google Drive):** חיבור בלחיצה אחת דרך Google OAuth — שמירה וטעינה אוטומטית.
- **ייצוא/ייבוא מקומי:** הורדת קובץ JSON וייבוא חזרה.

## מבנה המערכת

האפליקציה בנויה כולה כקובץ HTML יחיד (`index.html`) המכיל את כל קוד ה-HTML, ה-CSS וה-JavaScript.

| רכיב | טכנולוגיה |
|------|-----------|
| עיצוב | Tailwind CSS (CDN) |
| מפות | Leaflet.js + CartoDB tiles |
| גאוקודינג | Nominatim (OpenStreetMap) — ללא API key |
| ניווט | Google Maps Embed (legacy format, ללא API key) |
| AI | OpenAI API / Google Gemini API |
| אבטחה | Web Crypto API (AES-GCM), CSP, iframe sandbox |
| גיבוי | Google Drive API (OAuth 2.0, scope: drive.file) |
| אחסון מקומי | localStorage |

## אבטחה

- **XSS Protection:** כל נתון ממשתמש/AI עובר `esc()` לפני הזרקה ל-innerHTML, ו-`escAttr()` לתוך onclick attributes.
- **Content Security Policy:** מוגבל למקורות ידועים בלבד (scripts, connections, frames).
- **iframe Sandbox:** כל iframe מוגבל עם `sandbox` attribute.
- **API Keys:** Gemini key נשלח דרך `x-goog-api-key` header (לא ב-URL). מפתחות מוצפנים ב-AES-GCM ב-localStorage.
- **JSON Validation:** ייבוא קבצים עובר ולידציית מבנה. תשובות AI עוברות try-catch עם הודעת שגיאה מוגבלת.

## דרישות קדם והפעלה

האפליקציה רצה כולה בצד הלקוח (Client-Side):

1. פתחו את `index.html` בדפדפן מודרני.
2. להפעלת AI, יש צורך במפתח API של OpenAI או Google Gemini (הזינו בהגדרות ⚙️).
3. Google Drive sync עובד אוטומטית — לחצו "גיבוי" בסרגל העליון.

> **הערה:** לפיתוח מקומי עם Google Drive, יש להריץ שרת HTTP מקומי (למשל `python -m http.server 8080`) ולהוסיף `http://localhost:8080` ל-Authorized JavaScript origins ב-Google Cloud Console.

## מדריך למשתמש

1. **הגדרת מפתח AI:** לחצו ⚙️ → לשונית OpenAI/Gemini → הדביקו מפתח → שמרו.
2. **יצירת מסלול:** לחצו "🤖 AI מסלול" → הזינו יעד, מספר ימים ודגשים → המתינו.
3. **עריכה ידנית:** עיפרון ליד שם היום → עריכת יעד, תיאור, מלון ויעדים.
4. **שדרוג יומי:** כפתורים בתחתית כל יום (טיפים, אוכל, מלונות, אופציות, שירותים).
5. **ניווט:** כפתור "🗺️ מסלול בניווט" → הצגה בתוך האפליקציה (משמאל) או פתיחה בדפדפן.
6. **השוואת גרסאות:** סמל שעון ⏪ → "השווה במפה" → מסלול בניווט להשוואה ב-Google Maps.
7. **גיבוי:** כפתור "📥 ייצוא" לגיבוי מקומי, או "גיבוי" לענן.

## הערות למפתחים

- **Geocoding Limit:** Nominatim (OSM) עם תור והשהיה למניעת Rate Limiting.
- **Fallback API Key:** ניתן להצפין מפתח גיבוי דרך לשונית "כלים" בהגדרות ולהחליף `FALLBACK_OAI_KEY` בקוד.
- **Google OAuth Client ID:** מוגדר ישירות ב-`Config.CLIENT_ID`. להחלפה — עדכנו בקוד וב-Google Cloud Console.
