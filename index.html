🗺️ מסלול ניווט יומי

המסלול כולל את היעדים המרכזיים, אפשרויות והמלצות שבחרת.

ברוכים הבאים למתכנן הטיולים החכם! 🌍

1. הגדרת חיבור (חובה): עליך להזין מפתח API (של OpenAI או Gemini) בלשונית המתאימה, לבחור אותו כספק הפעיל למטה ולשמור.

2. תחבורה חכמה: הכפתור "🚕 תחבורה" ינתח את כל מסלול הטיול ויציע דרכי התניידות אופטימליות (טיסות, רכבות) שישובצו בין הימים.

3. ספריה והיסטוריה: כפתור 📂 פותח את ספריית הטיולים שלך. כפתור ⏪ (למעלה או בכל יום נפרד) יאפשר לשחזר ולהשוות גרסאות עבר על המפה!

4. שדרוג כל יום בנפרד (כפתורי תחתית):

  • ✨ טיפים: פנינים נסתרות ותצפיות.
  • 🍜 אוכל: מסעדות פעילות ומאכלים מקומיים.
  • 🏨 מלונות: אפשרויות לינה מובילות.
  • 🎯 אופציות: חלופות נוספות לפעילויות.
  • 👷 שירותים: ספקים מקומיים (מדריכים/נהגים) עם ביקורות גבוהות.
  • 🤖 תכנן מחדש: כתיבת הלו"ז היומי מחדש על בסיס בקשתך.

שינוי הפרומפטים עלול לשבור את מבנה ה-JSON שמייצרת המערכת.
