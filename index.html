🗺️ מסלול ניווט יומי

המסלול כולל את היעדים המרכזיים, אפשרויות והמלצות שבחרת.

ברוכים הבאים למתכנן הטיולים החכם! 🌍

1. הגדרת חיבור (חובה): כדי שה-AI יעבוד, עליך להזין מפתח API (של OpenAI או Gemini) בלשונית המתאימה, לבחור אותו כספק הפעיל למטה ולשמור.

2. בניית מסלול מהיר: לחץ על "בניית מסלול AI" במסך הראשי, הזן את היעד ומספר הימים, והמערכת תבנה עבורך מסלול שלם.

3. שדרוג כל יום בנפרד: בכל יום במסלול ישנם כפתורי קסם בתחתית הכרטיסייה:

  • ✨ טיפים: קבלת המלצות למקומות סודיים ותצפיות סביב יעד היום.
  • 🍜 אוכל: המלצות למסעדות ספציפיות פעילות באזור.
  • 🎯 אופציות: יצירת חלופות נוספות לפעילויות באותו יום.
  • 🤖 תכנן מחדש: מחיקת לו"ז היום הקיים וכתיבתו מחדש לפי בקשה ספציפית שלך.

4. ניווט חכם: כפתור "🗺️ נווט" אוסף את היעדים, הפעילויות והטיפים שבחרת ביום מסוים, ומייצר מסלול נסיעה אוטומטי ב-Google Maps.

5. גיבוי ושמירה: התחבר עם חשבון גוגל ("גיבוי בענן") כדי לשמור את המסלול אוטומטית ב-Google Drive שלך, או השתמש ב"ייצוא" לשמירת קובץ מקומי במחשב.

שינוי הפרומפטים עלול לשבור את מבנה ה-JSON שמייצרת המערכת.
